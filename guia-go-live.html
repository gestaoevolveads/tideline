PASSO 4
Publicar o site (Cloudflare Pages)
Como o repo é privado, hospedamos no Cloudflare (grátis). Em Workers & Pages → Create → Pages → Connect to Git → escolher gestaoevolveads/tideline. Nas build settings: Framework = None, Build command vazio, Build output directory = demo → Save and Deploy. Sai uma URL tipo tideline.pages.dev.
Abrir o Cloudflare ↗

PASSO 5
